## Utilizando a aplicação

### Frontend
- Navegue até a pasta `frontend`
- Instale as dependências: `npm install`
- Inicialize o servidor frontend: `npm run dev`

### Backend
- Navegue até a pasta `backend`
- Instale as dependências: `npm install`
- Inicialize o servidor frontend: `npm run start`
